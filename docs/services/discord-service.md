# Discord Service

The Discord Service sends direct messages (DMs) to students on Discord.

## Features

- Reads `discord_messages.csv` generated by the GradeSync service.
- Resolves Discord User IDs from usernames if needed (though IDs are preferred).
- Sends DMs using a Discord Bot.
- Handles rate limiting automatically.

## Configuration

Required environment variables in `.env.local`:

```bash
DISCORD_BOT_TOKEN="your-bot-token"
DISCORD_GUILD_ID="your-server-id"
```

## Usage

1.  Ensure `discord_messages.csv` exists (generated by `db_fetch.py --discord-csv`).
2.  Run the script:

```bash
python3 services/discord_service/send_discord_reminders.py
```

## CSV Format

The service expects a CSV with the following columns:
- `discord_id`: The user's Discord ID (e.g., `123456789012345678`).
- `message`: The content of the message to send.
